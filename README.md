# gtaV-natives-header
This repo provides an updated and improved **natives.h** header file for making GTA5 <a href="http://www.dev-c.com/gtav/scripthookv/">ScriptHookV</a> C++ script mods.
This file was auto-generated using a python script and is based on <a href="https://github.com/alloc8or/gta5-nativedb-data/blob/master/natives.json">alloc8or's natives.json</a> file. In order for this header file to work correctly, you will also need the updated **nativeCaller.h** header from this repo.
